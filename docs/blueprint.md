## Core Features:

- Database Design: Define Firestore data structure and rules for courses, sections, lessons, and user progress.
- User Authentication: Implement user registration and login with email and password using Firebase Authentication.
- List Courses: Develop a Cloud Function to retrieve and list all active courses with basic details.
- Get Course Details: Implement a Cloud Function to get detailed information about a specific course, including sections and lessons.
- Mark Lesson Complete: Develop a Cloud Function to mark a lesson as completed by a user, storing the progress in Firestore.
- Get User Progress: Create a Cloud Function to retrieve a user's progress for a specific course.
- Basic UI: Simple UI that reflects the response from the calls to Firebase in React

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to evoke trust and stability.
- Background color: Light Blue-Gray (#ECEFF1) to provide a calm, neutral backdrop.
- Accent color: Vibrant Orange (#FF5722) for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, machined, objective, neutral look suitable for both headlines and body text.
- Use flat, line-based icons for course categories and actions.
- Employ a clean, card-based layout for course listings and detail pages.
- Subtle transitions and loading animations to enhance user experience.
